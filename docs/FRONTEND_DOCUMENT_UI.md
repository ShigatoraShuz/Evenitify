# Frontend Document UI

Document UI is frontend-only in Phase 6.

## Components

- `ContractDocumentCard`
- `UploadDocumentDropzone`
- `AttachmentList`
- `DocumentPreviewModal`

## States

- `not_uploaded`
- `uploaded`
- `pending_review`
- `approved`
- `rejected`

Mock upload stores metadata only. Uploaded files are not stored locally, sent to a backend, or committed.
